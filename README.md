# Блогикум (Blogicum)

## О проекте
Блогикум — это простой Django-проект для ведения блога. Здесь пользователи могут создавать посты. 

## Ззапуск
1. Клонируй репозиторий:  
   `git clone https://github.com/JuliaDJ1/django-sprint1.git`
2. Перейди в папку:  
   `cd django-sprint1`
3. Создай виртуальное окружение:  
   `python -m venv venv`
4. Активируй:  
   - Windows: `venv\Scripts\activate`
5. Установи зависимости:  
   `pip install -r requirements.txt`
6. Запусти:  
   `python manage.py runserver`
7. Открой в браузере: http://127.0.0.1:8000/
